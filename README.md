# Project "FLASK blog"

## Description

Best blog at Flask.

## Installation

pip install -r requirements.txt

## Run

cd app/blogengine

./manage.py runserver


## Urls

http://127.0.0.1:5000/blog/

http://127.0.0.1:5000/login
